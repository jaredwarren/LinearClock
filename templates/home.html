iCal Sync
When enabled, configd will periodically fetch the iCal URL, expand recurring events, and regenerate Tick events (manual edits will be overwritten).

Lookback = how many days before now the sync window starts; Lookahead = how many days after now it ends (same time of day as “now”, not midnight). Use at least 1 lookahead to include upcoming events (e.g. weekday 10:00–10:30). 0 + 0 means the window is only the current instant—almost no events match.
